Gets the content of history section of the page

Displays top N keywords which occur the most in a particular section.

Instructions to use:

1) Build the app and run it.
2) Enter a numeric value that represents how many top keywords you want to see.
3) Press Y if you want to remove certain keywords from the result or Press N to skip this step.
	-If you press Y: Provide a string of words separated by space that you would like to remove.
4) The application would display the results. If you want to quit press Y
